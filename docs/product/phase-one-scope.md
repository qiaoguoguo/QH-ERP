# 一期范围与里程碑

## 一期目标

一期目标是建立可持续扩展的 ERP 基础能力，并优先打通生产管理核心链路所需的前置能力。

## 建议里程碑

1. 项目工程骨架与账号权限基础。
2. 基础资料与物料管理。
3. BOM 管理。
4. 仓库与库存基础能力。
5. 生产工单、领料、报工、完工入库。
6. 成本归集基础记录。

## 一期重点

- 建立统一物料、单位、仓库、BOM、工单状态口径。
- 保证库存变化和生产单据可追溯。
- 保证每个阶段成果可部署、可浏览器验收。

## 一期暂不深入

- 完整财务总账体系。
- 高级排产和复杂产能优化。
- 多公司、多组织、多租户能力。
- 零售收银、电商交易、门店会员能力。

## 决策待确认

- 首个开发模块是否确认为账号与权限基础。
- 成本归集一期做到业务记录还是可核算结果。
- 一期验收按模块逐步验收，还是最终补充一次端到端生产闭环验收。
