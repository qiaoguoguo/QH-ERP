# 任务记录：销售管理基础模块

## 任务名称

销售管理基础模块。

## 任务目标

建立制造业 ERP 一期之后的销售管理基础能力，让有权限用户能够维护销售订单，并基于销售订单完成销售出库。销售出库过账后必须扣减库存余额、生成库存流水，并能从销售订单、销售出库和库存流水之间追溯来源。

本阶段只做销售业务基础和库存出库追溯，不做应收、发票、收款、税务、正式收入确认、审批、MRP、销售退货、发运物流、报表看板或正式财务凭证。

## 产品口径

- 本阶段销售订单明细只允许选择启用的 `FINISHED_GOOD` 与 `SEMI_FINISHED` 物料。
- 启用的 `RAW_MATERIAL` 和 `AUXILIARY` 在本阶段视为不可销售物料，创建、更新或确认销售订单时必须拒绝。
- 本阶段不新增物料“可销售”字段，不扩大物料主数据范围。
- 销售订单创建、更新、确认时客户必须存在且启用。
- 销售订单确认后客户若被停用，不阻止基于该已确认或部分出库订单创建、更新或过账销售出库。
- 销售出库创建、更新和过账仍必须校验订单状态、来源行、物料、单位、仓库和库存；物料必须仍启用且仍属于 `FINISHED_GOOD` 或 `SEMI_FINISHED`，单位必须仍启用且仍为物料基本单位，仓库必须仍启用，库存必须充足。
- 销售单价只作为销售业务记录字段，不形成应收、收入确认、税额、成本结转或正式财务凭证。
- 本阶段统一使用物料基本单位，不做销售包装单位或单位换算。
- 已过账销售出库不可编辑，不支持反过账；错误修正后续通过库存调整并保留说明。
- 销售出库库存不足必须拒绝，不能允许负库存。
- 销售订单和销售出库作为独立销售域能力，不并入库存单据或生产域。
- 销售出库相关契约命名统一使用 `SALES_SHIPMENT`、`sales:shipment:*` 和 `/sales/shipments`，不使用 outbound 命名。

## 验收标准

- 功能验收：
  - 管理员能够创建销售订单草稿、编辑草稿、确认订单、取消订单和关闭订单。
  - 管理员能够基于已确认或部分出库订单创建销售出库草稿。
  - 管理员能够过账销售出库，过账后出库单不可编辑且不可重复过账。
  - 销售订单详情显示出库记录、已出库数量和未出库数量。
  - 销售出库详情显示来源订单和过账前后库存。
  - 库存流水显示销售出库来源，并能追溯到销售出库。
- 数据验收：
  - 销售订单草稿、确认、取消和关闭不改变库存余额。
  - 销售出库草稿不改变库存余额，不生成库存流水。
  - 销售出库过账后，库存余额扣减量等于本次出库数量。
  - 每条销售出库明细生成一条可追溯库存流水。
  - 同一出库明细不得重复生成库存流水。
  - 订单行已出库数量不得超过销售数量。
  - 库存不足时必须拒绝过账，不允许负库存。
  - 异常失败不得产生部分库存余额、部分流水或错误累计出库数量。
- 权限验收：
  - 管理员具备销售订单和销售出库全量权限。
  - 销售角色可维护销售订单并查看销售出库，不自动获得销售出库过账权限。
  - 仓库角色可查看订单必要信息，并维护、过账销售出库，不可确认、取消或关闭销售订单。
  - 只读用户只能查看销售订单、销售出库和追溯信息。
  - 无权限用户不能访问销售页面或接口。
  - 前端隐藏按钮不能替代后端接口鉴权。
- UI 验收：
  - 页面符合后台 ERP 信息密度和现有设计风格。
  - 销售订单列表、详情、表单和销售出库列表、详情、表单状态清晰。
  - 销售出库与库存调整出库文案不混淆。
  - 桌面和窄屏视口下关键操作可触达，密集表格可通过可信横向滚动查看。
  - 表单错误、异常提示、只读状态和无权限状态清晰可见。
- 测试验收：
  - 后端全量测试通过。
  - 前端全量测试、类型检查和构建通过。
  - 浏览器主路径、权限路径、异常路径和视觉分析完成。

## 风险与处理

- 库存扣减一致性：销售出库必须复用库存域受控过账能力，避免销售出库直接修改库存余额。
- 库存不足或并发出库：过账时必须在事务内校验现存数量并锁定相关余额，库存不足拒绝过账。
- 超出库风险：过账时锁定销售订单行，事务内计算剩余未出库数量并回写累计。
- 销售单价被误解为财务入账：接口文档、页面文案和测试结论均说明销售单价只是业务字段。
- 销售出库与库存单据混淆：销售页面和库存流水来源明确显示“销售出库”。
- 客户状态口径混淆：销售订单创建、更新、确认必须校验客户启用；订单确认后客户停用不阻断销售出库，但出库仍校验订单状态、来源行、物料、单位、仓库和库存。
- 可销售物料口径混淆：本阶段只允许 `FINISHED_GOOD` 和 `SEMI_FINISHED`，不得通过新增物料字段临时表达可销售属性。
- 视觉证据不足：阶段验收必须保存真实浏览器截图和视觉分析结论。
