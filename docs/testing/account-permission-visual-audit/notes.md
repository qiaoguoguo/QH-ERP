## 视觉分析发现与处理

- 发现：桌面用户列表中较长账号在下划线附近断行，导致行高被撑大。
  - 处理：用户表长文本列开启溢出提示，通用表格单元格禁止自动断词。
  - 状态：已修复并重新截图。
- 发现：390px 窄屏下侧边栏挤压主内容，用户页不可用。
  - 处理：调整样式导入顺序，确保项目样式覆盖 Element Plus；小屏布局改为顶部导航区，主内容宽度恢复为 100%。
  - 状态：已修复并重新截图。
- 发现：390px 窄屏下固定操作列遮挡状态、角色等业务列。
  - 处理：小屏取消固定列 sticky 行为，使用表格横向滚动查看宽列。
  - 状态：已修复并重新截图。
