# BOM 管理测试计划

## 目标

验证 BOM 管理阶段满足功能、接口、权限、异常、审计、部署和浏览器视觉验收要求，确保后续生产工单可以引用稳定、可追溯的 BOM 主数据。

## 准入标准

- 任务文档、设计规格、接口契约、权限编码和验收路径已确定。
- 账号权限模块、基础资料与物料管理在主分支可用。
- 本地 PostgreSQL、后端、前端启动方式可用。
- 验收物料、单位和分类测试数据口径已明确。

## 准出标准

- 前端 `npm test`、`npm run typecheck`、`npm run build` 通过。
- 后端 Docker 化 Maven/Testcontainers 测试通过。
- 管理员浏览器验收路径通过。
- 只读与越权接口验证通过。
- 视觉分析截图与结论已保存到 `docs/testing/bom-management-visual-audit/`。

## 测试角色

| 类型 | 用途 |
|---|---|
| 管理员 | 完成 BOM 新增、查询、详情、编辑草稿、复制版本、启用、停用和审计验收 |
| 只读用户 | 验证菜单可查看但写按钮不可见，后端写接口返回 `AUTH_FORBIDDEN` |
| 无权限用户 | 验证无菜单入口、路由受限和接口拒绝 |

## 测试数据

| 数据 | 口径 |
|---|---|
| 成品 A | 已启用物料，`materialType = FINISHED_GOOD`，`sourceType = SELF_MADE` |
| 半成品 B | 已启用物料，`materialType = SEMI_FINISHED`，`sourceType = SELF_MADE` |
| 原材料 X | 已启用物料，`materialType = RAW_MATERIAL`，`sourceType = PURCHASED` |
| 原材料 Y | 已启用物料，`materialType = RAW_MATERIAL`，`sourceType = PURCHASED` |
| 辅料 Z | 已启用物料，`materialType = AUXILIARY`，`sourceType = PURCHASED` |
| 停用原材料 T | 已停用物料，用于引用失败验证 |
| 单位 | 至少包含个、千克、米，均为启用状态 |

## 核心用例

- BOM 列表、查询、分页、详情。
- BOM 新增草稿、编辑草稿、明细新增、明细删除、明细排序。
- BOM 编码重复、版本重复、必填为空、状态非法、ID 不存在。
- 明细为空、用量为 0、用量为负、损耗率小于 0、损耗率大于等于 1。
- 父项物料不存在、停用、类型不允许或来源属性不允许。
- 子项物料不存在、停用、重复或等于父项。
- 基准单位不存在或停用。
- 明细单位不存在或停用。
- 直接循环引用和多级可检测循环引用。
- 启用 BOM、停用 BOM、同父项启用版本唯一。
- 复制 BOM 为新版本草稿，明细完整复制且新版本唯一。
- 启用和停用操作写入审计日志。
- 只读角色按钮不可见，后端越权返回 `AUTH_FORBIDDEN`。

## 功能测试范围

- 列表：按关键词、父项物料、状态查询，分页和空状态正确。
- 详情：展示主表、明细、审计时间和只读状态。
- 创建：填写父项、版本、基准数量、有效日期和明细后保存草稿。
- 编辑：草稿可修改主表和明细，启用或停用版本只读。
- 明细：新增行、删除行、选择物料、填写用量、损耗率和备注。
- 复制：从草稿、启用或停用 BOM 复制为新版本草稿。
- 启用：有明细且校验通过后变为 `ENABLED`。
- 停用：启用或草稿可停用，停用后不作为默认可选。

## 接口测试范围

- 未登录访问受保护接口返回 `AUTH_UNAUTHORIZED`。
- 未登录访问列表和详情接口返回 `AUTH_UNAUTHORIZED`。
- 无 BOM 查看权限访问列表和详情接口返回 `AUTH_FORBIDDEN`。
- 无权限访问写接口返回 `AUTH_FORBIDDEN`。
- 列表接口返回 `PageResponse<T>`。
- 详情接口在 ID 不存在时返回 `BOM_NOT_FOUND`。
- 写接口校验 CSRF token。
- 创建请求不传 `status` 时后端默认保存为 `DRAFT`。
- 更新请求只允许草稿状态，启用和停用状态返回 `BOM_STATUS_NOT_EDITABLE`。
- 创建、更新、复制校验 BOM 编码全局唯一。
- 创建、更新、复制校验同父项版本唯一。
- 启用接口校验同父项唯一启用版本。
- 创建、更新、启用校验父项物料存在、启用、类型和来源属性正确。
- 创建、更新、启用校验子项物料存在且启用。
- 创建、更新、复制、启用校验基准单位存在且启用。
- 创建、更新、复制、启用校验明细单位存在且启用。
- 创建、更新、启用校验明细不能为空。
- 创建、更新、启用校验数量和损耗率范围。
- 创建、更新、启用校验父项不能等于子项。
- 创建、更新、启用校验同一 BOM 内子项不能重复。
- 创建、更新、启用校验直接循环和多级可检测循环。
- 创建、更新、复制、启用、停用写入审计日志。

## 权限测试范围

- 管理员具备 BOM 查看、创建、更新、复制、启用、停用权限。
- 只读用户具备查看权限时可进入列表和详情。
- 只读用户没有创建、编辑、复制、启用、停用按钮。
- 只读用户直接调用创建、更新、复制、启用、停用接口返回 `AUTH_FORBIDDEN`。
- 无权限用户不能通过直接路由访问 BOM 页面。
- 权限变更以后端实时鉴权为准，前端刷新当前用户或重新登录后更新展示。

## 浏览器验收路径

1. 管理员登录。
2. 进入物料管理下的 BOM 管理。
3. 查看列表默认状态和空状态。
4. 新建成品 A 的 `V1.0` BOM 草稿，添加原材料 X、半成品 B 和辅料 Z 明细。
5. 验证必填为空、重复版本、重复子项、用量错误、父项等于子项和停用物料引用错误。
6. 启用 `V1.0` BOM，查看详情只读状态。
7. 复制 `V1.0` 为 `V1.1` 草稿，调整用量后尝试启用，验证同父项启用版本唯一错误。
8. 停用 `V1.0` 后启用 `V1.1`。
9. 构造半成品 B 的 BOM 指向成品 A，验证可检测循环引用被拒绝。
10. 只读用户登录验证权限表现。
11. 保存桌面和窄屏视觉截图。

## 视觉分析要求

- 浏览器验收阶段必须保存关键页面截图和分析结论，证据目录使用 `docs/testing/bom-management-visual-audit/`。
- 截图至少覆盖桌面视口和一个窄屏视口。
- 关键页面截图必须覆盖 BOM 菜单入口、列表默认状态、查询结果、空状态、创建表单、明细编辑表格、表单校验错误、循环引用或重复版本错误、详情只读状态、复制版本、启停确认、只读权限状态和窄屏明细表格。
- 布局稳定性检查：页面标题、查询区、表格、分页、抽屉、弹窗、明细表格和操作区不能在加载、空状态、错误状态或权限状态切换时出现异常跳动。
- 信息密度检查：桌面端应保持 ERP 后台可扫描的信息密度，BOM 列表和明细表格行高、列宽和筛选区间距不能明显降低维护效率。
- 关键操作可见性检查：新增、编辑、复制、启用、停用、保存、取消、查询和重置操作在有权限时清晰可见，在无权限时不可误导用户。
- 文案溢出检查：BOM 编码、物料编码、版本、物料名称、错误提示和明细备注不能穿出容器或遮挡相邻内容。
- 控件重叠检查：表格固定列、横向滚动、弹窗底部按钮、下拉框和提示信息不得互相遮挡。
- 权限状态检查：只读视图写按钮不可见，越权或无权限反馈可识别。
- 异常状态检查：重复编码、重复版本、停用物料引用、循环引用、必填为空、列表为空和加载失败提示清楚，不应关闭未保存表单。
- 如发现影响核心操作、关键数据识别、权限状态判断或工作台扫描效率的视觉问题，必须修复并重新截图后才能进入阶段验收。

## 回归范围

- 登录、退出和当前用户权限。
- 菜单渲染、路由守卫和按钮权限。
- 基础资料与物料管理中的单位、物料分类、物料档案列表和查询。
- BOM 列表、表单、详情、启停、复制和错误反馈。
- 后端接口鉴权和审计日志。
- 数据库迁移和权限种子。
- 窄屏导航、查询区、表格横向滚动、弹窗和抽屉可用性。

## 阻断缺陷

- 本地服务无法启动或浏览器无法访问核心页面。
- 管理员无法完成 BOM 新增、编辑、启用、复制或停用。
- BOM 编码唯一约束失效。
- 同父项版本唯一约束失效。
- 同父项多个启用版本。
- 循环引用可保存或启用。
- 启用 BOM 无明细。
- BOM 可引用不存在或停用的父项、子项或单位。
- 无权限用户可绕过后端接口鉴权。
- 写操作没有审计记录。
- 关键页面在桌面或窄屏下主内容不可用、关键按钮不可见、明细列互相遮挡或文本严重错位。
