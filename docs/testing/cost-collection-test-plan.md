# 成本归集基础测试计划

## 目标

验证成本归集基础阶段满足功能、接口、权限、异常、来源追溯、审计、部署和浏览器视觉验收要求，确保成本记录能够可信承接生产领料、报工和完工入库结果，同时不被误用为正式财务核算结果。

## 准入标准

- 任务文档、设计规格、接口契约、权限编码和验收路径已确定。
- 账号权限、基础资料与物料管理、BOM 管理、仓库与库存基础能力、生产执行基础能力在当前分支可用。
- 本地 PostgreSQL、后端、前端启动方式可用。
- 已确认成本归集一期只做业务记录和追溯。
- 已确认自动材料来源记录不自动计算材料金额。
- 已确认完工入库作为产出数量追溯，不自动形成正式成本金额。

## 准出标准

- 前端 `npm test`、`npm run build` 通过。
- 后端 Maven/Testcontainers 测试通过。
- 管理员浏览器验收路径通过。
- 成本管理角色、只读用户和无权限用户权限验收通过。
- 后端越权接口验证通过。
- 视觉分析截图与结论已保存到 `docs/testing/cost-collection-visual-audit/`。
- `git diff --check` 无输出。

## 测试角色

| 类型 | 用途 |
|---|---|
| 管理员 | 完成生产来源准备、成本自动记录、手工记录、查询、详情、追溯和审计验收 |
| 成本管理角色 | 验证成本记录查看、创建和更新权限，不额外拥有无关生产库存写权限 |

| 无权限用户 | 验证无菜单入口、路由受限和接口拒绝 |

## 后端接口测试范围

- 未登录访问成本记录接口返回 `AUTH_UNAUTHORIZED`。
- 无对应权限访问成本查询和写接口返回 `AUTH_FORBIDDEN`。
- 写接口缺少 CSRF token 被拒绝。
- 成本记录列表按关键词、工单、产品、成本类型、来源类型、来源单号和日期范围筛选分页。
- 成本记录详情返回工单、产品、来源单据、成本类型、金额或数量口径、记录人和记录时间。
- 生产领料过账后自动生成材料来源成本记录，来源明细、物料、数量、单位、工单、产品一致。
- 生产报工过账后自动生成人工数量口径成本记录，数量与报工单一致。
- 完工入库过账后工单成本汇总可展示产出数量追溯。
- 重复过账或重复生成同一来源不产生重复成本记录。
- 手工成本记录创建成功，支持人工、制造费用、其他成本和必要材料金额补录。
- 手工记录金额为空、负数、超精度，数量为空、负数、超精度时返回明确错误。
- 手工记录关联不存在工单、已取消工单或无效产品时返回明确错误。
- 自动生成记录不可修改来源工单、产品、来源单据和来源数量。
- 成本记录写入不得改变库存余额、生产工单状态或生产单据状态。
- 所有写操作写入审计日志。

## 前端测试范围

- 路由权限：无权限访问成本页面跳转无权限页。
- 成本记录列表：筛选、重置、分页、加载失败、空状态和权限按钮显示。
- 成本记录详情：工单、产品、来源单据、成本类型、金额、数量、记录人、记录时间和追溯入口展示。
- 手工记录表单：必填校验、金额校验、数量校验、保存失败提示、防重复提交。
- 手工记录编辑：编辑模式、保存成功、保存失败和自动记录不可编辑。
- 生产工单详情：成本归集记录分区展示，能进入成本记录详情。
- 只读权限：无创建、更新或生成按钮。
- 窄屏视口：成本列表、详情和表单可访问，关键操作不重叠。

## 浏览器验收路径

1. 管理员登录。
2. 准备启用物料、BOM、仓库和库存。
3. 创建成品 A 的生产工单草稿并发布。
4. 创建并过账生产领料，确认材料成本来源记录生成。
5. 创建并过账生产报工，确认人工数量口径记录生成。
6. 创建并过账完工入库，确认工单成本汇总显示产出数量追溯。
7. 在成本记录列表按工单、产品、成本类型、来源类型和日期范围查询。
8. 进入材料成本记录详情，确认来源领料单、来源明细、工单、产品、物料、数量和记录人。
9. 进入人工成本记录详情，确认来源报工单、报工数量、工单、产品和记录人。
10. 从成本记录详情追溯回生产工单。
11. 从生产工单详情查看成本归集记录分区并进入成本详情。
12. 手工录入一条制造费用或其他成本记录，关联同一工单。
13. 编辑该手工成本记录，确认金额、数量、业务日期和说明可受控更新。
14. 成本管理角色登录，验证可查看、创建和更新成本记录，但不能执行无关生产和库存写操作。
15. 尝试重复生成或重复来源归集，确认返回受控错误。
16. 尝试非法金额、非法数量、无效工单或草稿来源，确认返回受控错误。
17. 只读用户登录，验证成本记录可看、写按钮不可见、写接口返回 `AUTH_FORBIDDEN`。
18. 无权限用户访问成本页面和接口，验证返回认证或鉴权错误。
19. 保存桌面和窄屏视觉截图。

## 阻断标准

- 本地服务无法启动或浏览器无法访问核心页面。
- 管理员无法完成成本归集主路径。
- 生产过账后成本来源记录缺失。
- 成本记录无法追溯到工单、产品或来源单据。
- 成本金额、数量口径错误，或来源单据与成本记录不一致。
- 重复来源导致重复成本记录且无受控错误。
- 成本记录反写库存余额、生产工单状态或生产单据状态。
- 无权限用户可绕过后端接口鉴权。
- 写操作无审计或成本链路不可追溯。
- 页面或文案暗示正式财务成本核算已经完成。
- 浏览器视觉分析缺失，或视觉问题导致核心数据、关键按钮或权限状态无法识别。

## 阶段排除

本阶段不测试正式成本核算、总账、凭证、标准成本、移动加权、先进先出、成本差异分析、制造费用自动分配、采购销售联动、多币种、多组织、多公司、多租户或 BI 报表。
