## 功能验收结果

- 管理员完成基础资料数据准备：`PCS / 件`、`WH-A / 一号仓`、`SUP-A / 原料供应商`、`CUS-A / 成品客户`。
- 管理员完成物料分类准备：`RAW / 原材料`、`SEMI / 半成品`、`FIN / 成品`、`AUX / 辅料`。
- 管理员完成物料档案准备：
  - `MAT-RAW-001 / 冷轧钢板-已编辑 / 原材料 / 外购 / 启用`。
  - `MAT-SEMI-001 / 半成品件 / 半成品 / 自制 / 停用`。
  - `MAT-FIN-001 / 整机 / 成品 / 自制 / 启用`。
  - `MAT-AUX-001 / 外协辅料 / 辅料 / 委外 / 启用`。
- 重复创建 `MAT-RAW-001` 返回错误提示 `主数据编码已存在`，表单保持打开。
- 停用已被启用物料引用的 `PCS / 件` 返回错误提示 `计量单位已被启用物料引用`，单位状态保持启用。
- 使用无匹配关键词查询物料档案时，列表展示 `暂无物料档案数据`，分页总数为 0。
- 只读用户页面不显示新增、编辑、启用、停用按钮。
- 只读用户直接调用物料创建接口返回 `403 / AUTH_FORBIDDEN`。

## 视觉分析结论

- 菜单结构：基础资料与物料管理分组清晰，当前菜单高亮明确，满足 ERP 后台扫描效率要求。
- 查询区：桌面端筛选项密度适中，计量单位、物料档案等页面的查询和重置按钮位置稳定。
- 表格区：桌面端表格行高和列宽可读，编码、名称、规格、分类、单位、类型、来源、状态和操作列没有文本穿出容器。
- 物料状态：`MAT-SEMI-001` 停用标签与其它启用物料可区分，满足状态识别要求。
- 权限状态：只读用户仅有详情入口，写操作入口不可见，后端越权拒绝结果与前端表现一致。
- 异常状态：重复编码、停用引用和空列表状态均有页面级视觉证据；加载失败属于服务异常场景，本阶段不主动破坏部署服务制造截图，后续在接口错误注入或端到端异常测试中补充更完整的服务失败截图。
- 弹窗：验收中发现弹窗遮罩与层级样式缺失、长表单存在标题和底部按钮可见性风险，已补充 `el-overlay.css` 并增加全局弹窗最大高度、内部滚动和移动端宽度约束。复验后遮罩生效，底层表格不再穿透，标题和底部按钮稳定可见。
- 窄屏：390px 视口下菜单、筛选区和表格可操作，表格通过横向滚动承载多列信息；当前满足阶段验收，后续可在移动端进一步优化表格列展示密度。
