# 生产执行基础测试计划

## 目标

验证生产执行基础阶段满足功能、接口、权限、异常、库存一致性、审计、部署和浏览器视觉验收要求，确保工单、领料、报工、完工入库和库存追溯链路可信。

## 准入标准

- 任务文档、设计规格、接口契约、权限编码和验收路径已确定。
- 账号权限、基础资料与物料管理、BOM 管理、仓库与库存基础能力在主分支可用。
- 本地 PostgreSQL、后端、前端启动方式可用。
- 已确认一期不启用库存占用、不允许超领、不允许超报、不允许超入。
- 已确认工单发布时生成 BOM 快照。

## 准出标准

- 前端 `npm test`、`npm run build` 通过。
- 后端 Maven/Testcontainers 测试通过。
- 管理员浏览器验收路径通过。
- 生产人员、仓库人员、只读用户和无权限用户权限验收通过。
- 后端越权接口验证通过。
- 视觉分析截图与结论已保存到 `docs/testing/production-execution-visual-audit/`。

## 后端接口测试范围

- 未登录访问生产工单、领料、报工、完工入库接口返回 `AUTH_UNAUTHORIZED`。
- 无对应权限访问生产查询和写接口返回 `AUTH_FORBIDDEN`。
- 写接口缺少 CSRF token 被拒绝。
- 工单列表按关键词、状态、产品物料和日期范围筛选分页。
- 创建工单校验产品物料、BOM、计划数量、计划日期、领料仓库和入库仓库。
- 草稿工单可更新；已发布、生产中、已完成、已取消工单不可更新关键字段。
- 发布工单后生成 BOM 用料快照，应领数量计算正确。
- BOM 未启用、父项不匹配、BOM 明细为空、停用物料或停用仓库导致发布失败。
- 无已过账业务的工单可取消；已有领料、报工或入库记录的工单不可取消。
- 领料草稿创建、更新和过账成功。
- 领料过账后库存余额减少，库存流水包含 `PRODUCTION_ISSUE` 来源。
- 领料库存不足、累计领料超过应领数量、停用仓库或停用物料返回明确错误，余额和流水不变化。
- 报工草稿创建、更新和过账成功。
- 报工过账后更新工单累计报工、合格和不良数量，不生成库存流水。
- 报工数量为 0、负数或累计超过计划数量返回明确错误。
- 完工入库草稿创建、更新和过账成功。
- 完工入库过账后成品库存余额增加，库存流水包含 `PRODUCTION_RECEIPT` 来源。
- 完工入库数量超过累计合格报工数量返回明确错误，余额和流水不变化。
- 已过账领料、报工和完工入库不可编辑。
- 重复过账同一生产单据返回受控错误，不能重复影响库存。
- 并发领料同一物料时库存不得为负，成功流水与最终余额一致。
- 所有写操作写入审计日志。

## 前端测试范围

- 路由权限：无权限访问生产页面跳转无权限页。
- 生产工单列表：筛选、重置、分页、加载失败、空状态和权限按钮显示。
- 生产工单表单：必填校验、数量校验、BOM 选择、仓库选择、保存失败提示。
- 工单详情：状态标签、用料快照、领料记录、报工记录、入库记录、库存流水摘要和操作按钮。
- 领料页面：默认加载用料快照，本次领料数量校验、库存不足错误提示、重复提交禁用。
- 报工页面：合格数量、不良数量、累计超计划校验、提交失败提示。
- 完工入库页面：仓库必选、入库数量校验、超合格报工错误提示。
- 只读权限：无创建、更新、发布、过账、完成和取消按钮。
- 窄屏视口：工单列表、详情明细和生产表单可横向滚动，底部操作不重叠。

## 浏览器验收路径

1. 管理员登录。
2. 创建成品 A 的生产工单草稿，选择启用 BOM、计划数量、领料仓库和入库仓库。
3. 发布工单，确认生成 BOM 用料快照。
4. 从工单详情创建领料单并过账，确认原材料库存减少，库存流水可追溯到领料单和工单。
5. 创建报工单并过账，确认工单累计报工、合格和不良数量更新，库存不变化。
6. 创建完工入库单并过账，确认成品库存增加，库存流水可追溯到完工入库单和工单。
7. 完成工单，确认工单状态为已完成。
8. 尝试库存不足领料，确认返回库存不足且余额不变。
9. 尝试超领、超报、超入，确认返回受控错误。
10. 尝试用停用 BOM、停用物料或停用仓库参与业务，确认被拒绝。
11. 仓库人员登录，验证其可执行领料和完工入库过账，不可发布工单。
12. 生产人员登录，验证其可报工，不可过账库存单据。
13. 只读用户登录，验证工单和追溯可看，写按钮不可见。
14. 只读用户直接调用写接口，验证返回 `AUTH_FORBIDDEN`。
15. 无权限用户访问生产接口，验证返回认证或鉴权错误。
16. 保存桌面和窄屏视觉截图。

## 阻断标准

- 本地服务无法启动或浏览器无法访问核心页面。
- 管理员无法完成工单、领料、报工、完工入库和追溯主路径。
- 发布工单未生成 BOM 快照。
- 领料后库存未扣减，完工入库后库存未增加，或余额与流水不一致。
- 库存可被扣成负数。
- 报工或入库数量可突破本阶段规则。
- 无权限用户可绕过后端接口鉴权。
- 写操作无审计或生产链路不可追溯。
- 浏览器视觉分析缺失，或视觉问题导致核心数据、关键按钮或权限状态无法识别。

## 阶段排除

本阶段不测试高级排产、工艺路线、工序流转、批次、序列号、库位、条码、倒冲、退料、补料、返工、委外、采购销售联动、正式成本核算、多单位换算、多组织、多公司或多租户能力。
