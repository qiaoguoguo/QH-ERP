# 测试数据基线

## 目标

建立后续可重复验收的测试数据口径。

## 基础数据

- 用户与角色。
- 客户与供应商。
- 仓库。
- 单位。
- 物料分类。

## 生产数据

- 原材料。
- 半成品。
- 成品。
- BOM 版本。
- 初始库存。
- 生产工单。
- 领料记录。
- 报工记录。
- 完工入库记录。
- 成本归集记录。

## 数据原则

- 测试数据应能覆盖正常流程和异常流程。
- 核心数据应可重复初始化。
- 变更测试数据必须记录原因。
- 初始化数据必须包含可登录管理员账号、基础角色和基础权限。
- 权限验收至少包含管理员账号、普通业务账号、无权限账号和停用账号。
- 生产主链路数据必须能覆盖物料、BOM、库存、工单、领料、报工、完工入库和成本记录。
